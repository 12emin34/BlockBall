Join wia the forcefield join command [Hubgame]

After this setup players will be able to join games by running into the forcefield and typing /bb [team] afterwards

  • 1. Open the config.yml of BlockBall and scroll down to forcefield-join
  • 2. Set enabled to true . Click here to find out more about dynamic commands
  • 3. Reload or restart your server. This setting cannot be applied via the blockballreload command
  • 4. Join a game by typing /bbjoin [teamname] after you have run into the forcefield
