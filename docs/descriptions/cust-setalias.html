Set a name for the arena

After this setup your arena will get a additional name, so players can join by typing the name of the arena instead of the id.

  • 1. Open your arena menu
  • 2. Enter a name via /Settings/General settings/Set alias (BlockBall supports colors [&a, &4,...])
  • 3. Save your arena /Save arena
  • 4. Type the /blockball command and look if the name shows up in the arena-list
