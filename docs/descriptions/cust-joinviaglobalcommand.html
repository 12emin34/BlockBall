Join wia the global join command

After this setup players will be able to join games by typing /bbjoin [game-number/game-alias] [teamname]

  • 1. Open the config.yml of BlockBall and scroll down to global-join
  • 2. Set enabled to true . Click here to find out more about dynamic commands
  • 3. Reload or restart your server. This setting cannot be applied via the blockballreload command
  • 4. Open the chat menu via /blockball and start navigating by entering /bbc [number]
