Dynamic commands

BlockBall provides command settings a bit different than most other plugins. The author is the opinion that you should be in charge of deciding what the name and permission of the command is.

This is a sample setup how to change the global command setup from /bbjoin [game-id/game-alias] [team] to /football [game-id/game-alias] [team]. You can do the same with every defined command in the config.yml

  • 1. Open the config.yml of BlockBall and scroll down to global-join
  • 2. Make sure enabled is set to true and set command to football
  • 3. Reload or restart your server. This setting cannot be applied via the blockballreload command
  • 4. Try joining a game by entering /football [game-id/game-alias] [team]
