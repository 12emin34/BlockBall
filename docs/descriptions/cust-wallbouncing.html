Wallbouncing

A often requestet feature is that balls should "bounce" of walls. BlockBall has implemented such a thing but it requires some setup before it is activated.

After this setup balls will no longer get stuck in corners.

  • 1. Open your arena menu
  • 2. Open the menu /Settings/General settings/Wall bouncing
  • 3. Find out the block-id and block-damage value of the wall of your arena. If you have more than one type of block simply repeat the following step.
  • 4. Use the Add block-type to bouncing option and enter the id in the correct format
  • 5. Save your arena /Save arena
  • 6. Join the game and shoot the ball against a wall with the entered id and show if it bounces back from it.
