Scoreboard for goals

After this setup your players will obtain a scoreboard during a game where they can find the time left and amount of goals each team has scored.

  • 1. Open your arena menu
  • 2. Move to /Settings/Annoucements/Set scoreboard
  • 3. Use Toggle scoreboard to enable the scoreboard for your game
  • 4. [Optional] Change the title by using Set title and entering a custom title
  • 5. [Optional] To change a specific line use Edit lines/Select line and enter the number of the line you want to edit

    Use Set Text and enter your wanted text. The following placeholders are available:

    • :countdown - Time left
    • :red - Name of teamred
    • :blue - Name of teamblue
    • :redcolor - Color of teamred
    • :bluecolor - Color of teamblue
    • :redscore - Score of teamred
    • :bluescore - Score of teamblue
  • 6. Save your arena /Save arena
  • 7. Join the game and check if the scoreboard pops up
Scoreboard for stats

After this setup your players will obtain a scoreboard when the join the server which displays certain stats.

  • 1. Open the BlockBall config.yml
  • 2. Move to stats-scoreboard
  • 3. Set enabled to true and modify the other settings if necessary

    The following placeholders are available:

    • <player> - Name of the player
    • <winrate> - Winrate of the player
    • <playedgames> - Amount of played games
    • <goalspergame> - Amount of goals per game
  • 3. Restart your server. This setting cannot be applied via the blockballreload command
