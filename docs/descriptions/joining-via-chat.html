Joining by typing the teamname does not work?

It is often the case that some Chat-Plugins especially Bungeecord Chat-Plugins do not handle chat messages correctly to be used with BlockBall. Simply decided decide for one of the other joining options or uninstall the Chat-Plugin.

  • Joining via the forcfield join command [HubGame]
  • Joining via the global join command [HubGame/MiniGame]
  • Joining by clicking signs [HubGame/MiniGame]
