In order to use the BlockBall plugin on your server make sure you have installed the following plugins:

  • WorldEdit

It is also required to install Java 8 or later to use the plugin. If you are not capable of installing Java 8 you can also use BlockBall v3.7.0 which is compatible to Java 7, however using the latest version is highly recommend.

First steps

1. Check if there are no errors between the loading and the enabled message of BlockBall 2. Join in game and type /blockball 3. If the blockball menu pops up, you are ready to go!

Creating your first game

To quickly check if your server is capable of running BlockBall let's create a hub game. They are often used on hubs to let player play football during waiting or just to fill your server-spawn with fun. How to create hubgame.

If you are already an experienced BlockBall user, you can also take a look at the other BlockBall gamemodes.


Gamemodes

HubGame

Select this gamemode to create a BlockBall game where players can join and leave whenever they want. It is mostly placed in lobbies or at server-spawns to provide a fun lobby game for players.

How to create a hubgame
Event Game

Select this gamemode if you want to create a live game where the administrator sets a referee who can control the game and have a fixed list of players on both teams. It is only used for events as it needs an administrator the whole time.

How to create a eventgame
Minigame

Select this gamemode if you want to use BlockBall as full standalone minigame where players have to join a BlockBall lobby, select their teams and after the countdown finishes the game starts. The game ends when the countdown is over or one team has reached a specific amount of goals.

How to create a minigame
BungeeCord Minigame

This is the same gamemode as Minigame, however it setups allows you to link servers together. It is recommend to use this gamemmode if you are using multiple server connected with BungeeCord and want to use one server just for one standalone BlockBall game.

How to create a bungeecordminigame

Hub Game

So let's create a hubgame! This was the first gamemode added to BlockBall and was originally intended to be the only gamemode. However, because of the great support many other gamemodes where implemented.

Create the arena in 10 steps
1. Join the game and type /blockball.
2. Enter "1" into the chat to create a new arena.(Not working? Click here)
3. Wait until the displayed menu pops up.
4. Select the corners of the arena via the WorldEdit Axe (Rightclick and Leftclick
5. Enter "1" into the chat to set the corners.
6. Wait until the displayed menu pops up.
7. Select the corners of the red goal via WorldEdit and type "2".
8. Select the corners of the blue goal via WorldEdit and type "3".
9. Move to the center of the arena and type "4" to set the spawnpoint of the ball.
10. Type "8" to save and enable the arena.
Playing the hub game

The forcefield of the arena is already active, close your menu by typing "e". Run into the forcefield and the game asks you to enter the name of the team in chat to join the game. Click here if typing the teamname is not working.

After a player has entered the field, the ball spawns automatically and you can score for your team. You can play quick games together and after a certain amount of goals, the game resets the counter to 0. Click here if the ball does not spawn.

After the ball has been shot into a goal, every player in the game will get notified by a title-screen-message and the ball respawns. Click here if the goals gets not registered.

Customizing the hub game

There are many different options to customize every gamemode. Click here to go to the customizing sections.

To edit an exisiting hub game simply type /blockball, then type "2" and change the settings by using the options mentioned above. Don't forget to save the arena after editing!

Mini Game

So let's create a minigame! It's a bit more difficult than the hubgame because there are many additional steps as linking lobbies, countdown and team balance. But the first steps are the same as the hubgame. Go to your wanted arena.

1. Create an arena similar to a hub arena
1. Join the game and type /blockball.
2. Enter "1" into the chat to create a new arena.(Not working? Click here)
3. Wait until the displayed menu pops up.
4. Select the corners of the arena via the WorldEdit Axe (Rightclick and Leftclick
5. Enter "1" into the chat to set the corners.
6. Wait until the displayed menu pops up.
7. Select the corners of the red goal via WorldEdit and type "2".
8. Select the corners of the blue goal via WorldEdit and type "3".
9. Move to the center of the arena and type "4" to set the spawnpoint of the ball.
10. Type "8" to save and enable the arena.
2. Setup the lobby
11. Press "6" to get to the settings and then type 1 to open the general settings.
12. Enter "1" to enable the minigame mode
13. Now go back by typing "b" and open the minigamemode settings by pressing "4"
14. Move to your wanted lobby spawn and type "3"
15. Move to your wanted leaving spawn (after a match ends) and type "4"
3. Player joining

There are 2 different ways how to let players join the lobby and game. You can use one or both ways. They are described below.

Joining via commands
1. Open the ../BlockBall/config.yml file and check if the global-join and global-leave command are enabled.
2. Check if your players have got the permission "blockball.user".

3. Type /bbjoin 'arenaid' to join your game.

Help! I want my players to join games via the arena name.
Joining via signs
1. Open the arena/Settings/Minigamemode settings page.
2. Setup a join sign by typing "5".

3. Afterwards Rightclick on a already placed sign to convert it. Repeat these steps as many times as you want.

4. Test your game

Save your arena! Go to the arena editing page and press "8".

Join a game by using one of the methods mentioned above and look if everything works as expected.

BungeeCord Minigame

So let's create a bungeecord minigame! This mode should be used when BlockBall should take control over a whole server. You setup one server as lobby server and the other one is completely run by BlockBall.

1. Before you start (Both-Server)
1. Copy the BlockBall.jar on your game-server and on your lobby-server.
2. Start your bungeecord network, the game-server and the lobby-server.
3. Join your game-server.
2. Create an arena similar to a hub arena (Game-Server)
1. Join the game and type /blockball.
2. Enter "1" into the chat to create a new arena.(Not working? Click here)
3. Wait until the displayed menu pops up.
4. Select the corners of the arena via the WorldEdit Axe (Rightclick and Leftclick
5. Enter "1" into the chat to set the corners.
6. Wait until the displayed menu pops up.
7. Select the corners of the red goal via WorldEdit and type "2".
8. Select the corners of the blue goal via WorldEdit and type "3".
9. Move to the center of the arena and type "4" to set the spawnpoint of the ball.
10. Type "8" to save and enable the arena.
2. Setup the lobby (Game-Server)
11. Press "6" to get to the settings and then type 1 to open the general settings.
12. Enter "3" to enable the bungeecordminigame mode
13. Now go back by typing "b" and open the minigamemode settings by pressing "5"
14. Move to your wanted lobby spawn and type "3"
15. Configure your fall-back server when players are kicked out of the blockball server
4. Finish the game-server (Game-Server)
1. Open the ../game-server/plugins/BlockBall/bungeecord.yml file and set enabled to true
2. Save your arena by going to the arena menu and typing "8"
3. Restart your game-server (automatically, a restart script is highly recommend)
5. Prepare the lobby-server (Lobby-Server)
1. Open the ../lobby-server/plugins/BlockBall/bungeecord.yml file and set enabled to true
2. Open the ../lobby-server/plugins/BlockBall/bungeecord.yml file and replace connection: MINIGAME by connection: SIGN
3. Restart your lobby-server
6. Finish the lobby-server (Lobby-Server)
1. Join your lobby-server
2. Type /bbbungee 'game-server-name' and rightlick on a sign
3. If the sign displays join, the server are linked together and players can join by click on the sign.
7. Customizing

If you want to change the arena, open the arena file located at ../game-server/plugins/BlockBall/arena/arena_0.yml and set enabled to false.

There are many different options to customize every gamemode. Click here to go to the customizing section.
